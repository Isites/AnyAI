# 物流配送参考

|------|---------|
| SF | 顺丰速运 |

| STO | 申通快递 |

| JD | 京东物流 |
| EMS | 中国邮政速递 |
| DB | 德邦快递 |

## 物流状态说明

| 状态 | 说明 |
|------|------|
| PENDING | 待发货 |
| SHIPPED | 已发货 |
| IN_TRANSIT | 运输中 |
| OUT_FOR_DELIVERY | 派送中 |
| DELIVERED | 已签收 |
| EXCEPTION | 异常（延误/丢件） |
| RETURNED | 退回 |

## 异常处理参考

| 异常类型 | 处理建议 |
|---------|---------|
| 延误 | 联系快递公司催件，必要时申请赔偿 |
| 丢件 | 联系快递公司核实，必要时申请赔偿或补发 |
| 拒收 | 核实拒收原因，如有异常可重新发货 |
| 退回 | 联系用户确认是否需要重新发货 |
